# Frontend Guidelines (Binding)

This folder contains the binding frontend conventions for this repository.

- `guidelines.md`: Global frontend rules (layout, styles, forms, UX consistency)
- `components.md`: Reusable component catalog and preferred usage

## Scope

These rules apply to all frontend changes in:

- `app/src/app/**`
- `app/src/components/**`
- `app/src/lib/**` (frontend-facing logic)

If there is a conflict:
1. User instruction
2. `.cursor/rules/*.mdc`
3. `docs/frontend/*`
